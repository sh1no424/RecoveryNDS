# 1С: Автоматизация выделения НДС (Переход с УСН на ОСНО)

Данная внешняя обработка предназначена для автоматизации бухгалтерского и налогового учета при смене налогового режима с УСН (упрощенная система) на ОСНО (общая система) в конфигурации **1С:Бухгалтерия предприятия 3.0**.
## 🎯 Проблема и решение
При переходе с УСН на ОСНО необходимо «выделить» НДС из себестоимости товаров, которые остались на складе, чтобы в дальнейшем принять его к вычету.
Ручное создание сотен документов — очень нудно и сложно. Данная обработка автоматизирует поиск остатков и создание проводок.

## 🛠 Ключевые возможности
* Автоматический анализ остатков товаров на заданном счете (по умолчанию 41.01) из регистра «Расходы при УСН».
* Встроенная группировка (свертка) данных по партиям и номенклатуре, что исключает ошибки дублирования сумм при наличии нескольких строк в документе поступления.
* **Генерация документов:**
    * Автоматическое создание документа **«Операция (бухгалтерский учет)»**: перенос НДС со счета 41 на счет 19.
    * Автоматическое создание документа **«Отражение НДС к вычету»**: формирование записи для Книги покупок.

## 🚀 Инструкция по применению
### Подготовка
1. Обязательно сделайте бэкап базы данных перед запуском.
2. Убедитесь, что в «Учетной политике» установлен способ оценки МПЗ **«По ФИФО»**.
3. Переключите систему налогообложения организации на **ОСНО** с даты перехода.

### Запуск обработки
1. Выберите **Организацию** и **Дату** (последний день работы на УСН).
2. Укажите **Счет учета** (например, 41.01).
3. Нажмите **«Найти остатки»** — таблица заполнится уникальными записями по партиям. 
4. Нажмите **«Сформировать документы»** — обработка создаст необходимые документы и выведет отчет о количестве сформированных операций.

## 📋 Технические требования
* Конфигурация: 1С:Бухгалтерия 3.0 (любой актуальный релиз).
* Права доступа: Пользователь должен иметь права на создание и проведение документов «Операция (бухгалтерский учет)» и «Отражение НДС к вычету».
